## MySql的自我提升

### 加载数据库

- 在DataGrip工具中链接MySql数据库，并且运行sql脚本文件`myemployees`
- 等待脚本文件运行完成后，本地MySql中就创建了一个名为`myemployees`的数据库
- 里面有四张相关联的表，主要存储了一个模拟的跨国公司的员工信息（包括部门信息，姓名，工作ID，工资等等）
